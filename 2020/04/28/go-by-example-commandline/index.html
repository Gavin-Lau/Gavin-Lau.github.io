Test单元测试

命名规则

比如main.go的文件夹中有个Add函数
func Add(a,b int) int
需要建立这个函数的单元测试函数，则需要两步:

  1. 在同一个包目录下建立”{被测文件名}_test.go”，本例子是main_test.go
  2. 在步骤1建立的文件中，建立函数”Test{被测函数}(t *testing.T)”函数，本例子是TestAdd(t *testing.T)
